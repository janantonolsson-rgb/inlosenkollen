# Nets / Nexi

> **Typ:** Kortinlösare + betalväxel (PSP)  
> **Marknadsposition:** #1 inlösare i Norden sett till antal handlare (~140 000 nordiska företag)  
> **Webb:** [payments.nets.eu/sv-SE](https://payments.nets.eu/sv-SE)  
> **Moderbolag:** Nexi Group (Italien/Europa)

---

## Översikt

Nets (varumärke under Nexi Group) är en av Nordens mest etablerade betalningsaktörer. De erbjuder allt från betalterminaler i butik till Nexi Checkout för e-handel. Styrkan ligger i omnikanal-lösningar och lokal nordisk support dygnet runt.

**Styrkor:**
- Störst antal nordiska handlarkunder
- Komplett omnikanal (butik + online + mobil)
- 15+ betalningsmetoder i checkout
- Utbetalning inom ett dygn
- Svensk support dygnet runt
- Färdiga integrationer (Shopify, WooCommerce)

**Svagheter:**
- Ingen publicerad prislista – kräver offert
- Ofta månadsavgift utöver transaktionsavgift
- Onboarding via säljkontakt (ej self-service)

---

## Avgifter

Nets/Nexi publicerar **inte** standardpriser online. Prissättning är offertbaserad och beror på:

- Transaktionsvolym (antal och värde)
- Kortmix (debit/kredit/företag/internationellt)
- Val av produkt (ren inlösen vs. checkout-paket)
- Bransch och riskprofil
- Butik vs. online vs. omnikanal

### Typisk avgiftsstruktur (branschuppskattning)

| Post | Typiskt intervall |
|---|---|
| Månadsavgift | 0–500 kr (varierar kraftigt) |
| Transaktionsavgift (kort) | 0,8–2,5 % |
| Fast avgift per transaktion | 0–0,50 kr |
| Startavgift | Ofta 0 kr vid kampanj |
| Terminalhyra | Separat (om fysisk terminal) |

*Exakta priser kräver offert från Nets säljteam.*

### Jämförelse med publicerade konkurrenter

Baserat på branschrapporter och jämförelseguider ligger Nets typiskt **mellan** Swedbank Pay (volym) och Stripe (self-service) prismässigt, med den viktiga skillnaden att Nets ofta tar en **månadsavgift** som Stripe inte har.

## Setup & onboarding

### Tidslinje

```
Kontakt/offert → Avtal → Teknisk setup → Integration → Test → Live
                         Total: ca 1–3 veckor
```

| Steg | Tid | Beskrivning |
|---|---|---|
| 1. Offertförfrågan | 1–3 dagar | Formulär på nets.eu eller telefon |
| 2. Avtal & KYC | 3–7 dagar | Företagsverifiering |
| 3. Produktsättning | 2–5 dagar | Checkout/terminal konfigureras |
| 4. Integration | 1–5 dagar | Plugin eller API |
| 5. Go-live | 1 dag | Testtransaktioner |

### Produkter för e-handel

| Produkt | Beskrivning |
|---|---|
| **Nexi Checkout** | Allt-i-ett checkout, 15+ betalmetoder |
| **One-page Shop** | Inbäddad checkout på landningssida |
| **Nexi Commerce** | Enterprise omnikanal-plattform |
| **Ren inlösen** | Bara kortinlösen (koppla till egen PSP) |

### Teknisk integration

| Metod | Beskrivning |
|---|---|
| Nexi Checkout (plugin) | WooCommerce, Shopify – plug-and-play |
| Hosted checkout | Omdirigering till Nets betalsida |
| API | REST API för custom-lösningar |
| iFrame / pop-up | One-page shop-integration |

### Betalmetoder (checkout)

- Visa, Mastercard, Maestro
- Apple Pay, Google Pay
- Swish (Sverige)
- MobilePay (Danmark), Vipps (Norge)
- Klarna (via integration)
- Faktura/delbetalning (via partner)
- Lokala betalmetoder per marknad

## Lämplig för

- Företag med både fysisk butik och webshop
- Nordiska e-handlare som vill ha lokal support
- Företag som vill samla alla betalmetoder (kort, Swish, Klarna) hos en leverantör
- Medelstora till stora volymer med förhandlingsutrymme

## Mindre lämplig för

- Startup som vill ha transparent, publicerad prissättning
- Self-service onboarding (kräver säljsamtal)
- Enbart ren kortinlösen till lägsta pris (Swedbank Pay kan vara billigare vid volym)
